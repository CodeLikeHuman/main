Python Lesson 1: Εισαγωγή στη Python

Σε αυτό το μάθημα θα δεις πώς να γράψεις απλό κώδικα σε Python και να χρησιμοποιήσεις βασικά στοιχεία όπως μεταβλητές και συνθήκες.

Quiz: Τι κάνει το παρακάτω κομμάτι κώδικα;

if 10 > 5:
  print("Yes")
Η σωστή απάντηση είναι: Yes ✅

Sample Python Code

Αυτό το πρόγραμμα ζητά την ηλικία σου και εμφανίζει ένα διαφορετικό μήνυμα ανάλογα με την απάντηση.


age = int(input("Πόσων ετών είσαι; "))

if age >= 18:
    print("Μπορείς να συμμετάσχεις στο μάθημα!")
else:
    print("Δυστυχώς πρέπει να είσαι τουλάχιστον 18 ετών.")
